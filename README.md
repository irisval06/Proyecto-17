# Proyecto-17
Proyecto Final de Análisis de Datos
Este proyecto final integra todos los conocimientos adquiridos durante el programa de análisis de datos, incluyendo manipulación de datos, consultas SQL, análisis exploratorio y visualización de información. El objetivo es resolver un problema de negocio real a partir de datos estructurados y comunicar los hallazgos de manera clara y profesional.
El proyecto se compone de tres entregables principales: un notebook en Jupyter con el análisis completo, una presentación en formato PDF con los resultados más relevantes y, opcionalmente, un dashboard interactivo desarrollado en Tableau Public para la visualización de métricas clave.
Objetivo
Aplicar un enfoque integral de análisis de datos para extraer insights relevantes, responder preguntas de negocio y presentar conclusiones accionables que apoyen la toma de decisiones estratégicas.
Actividades realizadas
Importación, limpieza y exploración del conjunto de datos.
Análisis exploratorio para identificar patrones, tendencias y relaciones relevantes.
Aplicación de consultas SQL para la extracción y transformación de datos.
Cálculo de métricas clave según los objetivos del negocio.
Visualización de resultados mediante gráficos y, opcionalmente, dashboard interactivo en Tableau.
Elaboración de conclusiones basadas en evidencia para apoyar la toma de decisiones.
Preparación de una presentación ejecutiva con los hallazgos principales.
Herramientas utilizadas
Python (Pandas, NumPy)
SQL
Jupyter Notebook
Tableau Public (opcional)
Matplotlib / Seaborn (visualización de datos)
Resultados esperados
El proyecto permite demostrar la capacidad de transformar datos en información útil, identificar insights clave y comunicarlos de forma efectiva a audiencias técnicas y no técnicas, simulando un entorno real de trabajo como analista de datos.
